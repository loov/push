# push3

push3 is a collection of unofficial tools and libraries for Ableton Push 3.

## References

- [LogicX-Push2](https://github.com/zurie/LogicX-Push2) -- Python implementation of Push 2 as a Logic Pro controller
- [push3-python](https://github.com/ofou/push3-python) -- Python library for Push 3 with display protocol findings
